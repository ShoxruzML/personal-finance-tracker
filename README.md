# Personal Finance Tracker

A simple command-line tool to track income and expenses, built in Python.

## Features
- Add income and expense entries with amount, category, and date
- Input validation (numeric amounts, valid dates in YYYY-MM-DD format)
- Data persisted to JSON

## Tech
- Python 3
- datetime for date handling
- JSON for storage

## How to run
`bash
python week1.py
